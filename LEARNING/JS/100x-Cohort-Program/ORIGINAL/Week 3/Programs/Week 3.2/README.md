# Programs for Cohort class 3.2

## To run the programs

1. open terminal in this folder.
2. run `npm install`.
3. To run the file:
   a. **1. fetchAPI.html**: Install the Live-Server extension from vscode. Right click on the file, and open in Live Server.
   b. **2. Auth_Tut.js**: run `node "2. Auth_Tut.js"`.
   c. **3. DB_Tut.js**: run `node "3. DB_Tut.js"`.

> **Note:** In DB_Tut.js make sure to replace `process.env.DB_URL` with your **cluster's MongoDB URL** before running the file.
